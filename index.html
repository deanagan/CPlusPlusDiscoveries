Example Algorithm: std::find_if

Finding the first odd number in a sequence of numbers
                        
                            std::vector<int> n { 2, 4, 6, 9, 10 };

                            auto result_iter = std::find_if(begin(n), end(n), ???);

Before C++11, we had 2 common approaches

Using a stand alone/free function
                        
bool IsOdd(int n) {
  return n % 2 != 0;
}
                        
                        
                        
void printOdd() {
  std::vector<int>::iterator it; // no auto yet!
  it = std::find_if(n.begin(), n.end(), IsOdd);

  if (it != n.end()) {
    std::cout << *it;
  }
}

Or using a function object or functor

                        
struct IsOdd {
  bool operator()(int n) const {
    return n%2 != 0;
  }
};
                        
                        
                        
void printOdd() {
  std::vector<int>::iterator it; // no auto yet!
  it = std::find_if(n.begin(), n.end(), IsOdd());

  if (it != n.end()) {
    std::cout << *it;
  }
}

With Modern C++(C++11 and beyond), we now have lambdas.

                        
void printOdd() {
  auto it = std::find_if(begin(n), end(n), [] (int n) {
      return n%2 != 0;
  });

  if (it != n.end()) {
    std::cout << *it;
  }
}

Discovering Lambdas

“A lambda expression is a mechanism for specifying a function object. The primary use for a lambda is to specify a simple action to be performed by some function.” - Bjarne Stroustrup

Syntax

                        
// Minimal lambda
auto fn = [] { };

Syntax of Lambdas

                        
auto fn = [] /* the square bracket is a “lambda introducer” */ { };

Syntax of Lambdas

                        
auto fn = [/*anything that goes here is a “capture”*/] { };

Syntax of Lambdas

                        
auto fn = [/*“capture”*/]
 /* Everything in between is optional:
  (...) mutable throwSpec -> retType */
 { /* body */ };

Syntax of Lambdas

                        
/*A lambda can have a parameter specified in parentheses.
This is also called a lambda declarator.*/
auto fn = [](const std::string& name)
{ std::cout << "Hello, " << name << "\n"; };

Captures

                                    [=]{};
Outer scope is passed to the lambda by value default.
                                    [&]{};
Outer scope is passed to the lambda by reference default.
                                    [&i]{};
Capture i by reference.

Captures

                                    [i]{};
Capture i by value.
                                    [&i, j]{};
Capture i by reference and capture j by value.
