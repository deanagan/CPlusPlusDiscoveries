Coding with STL Algorithms

The C++ Standard Template Library

Created by Dean Agan

Why use the STL?

  • It is well tested and optimised.
  • Don't reinvent the wheel.
  • It leads to shorter more compact code.

STL Approach

  • There is preference for free functions over member functions (e.g. std::find(container..) over container.find() ).
  • Algorithms are designed to work for most containers.
  • Algorithms work with iterators.

Standard Template Library Categories

  • Collections (vectors, list, etc)
  • Algorithms (find, copy, sort, etc)
  • Iterators - common interfaces for any container type. (input, output, forward, bidirectional, random access iterators. C++17 has contiguous)
